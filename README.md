# TaskFlow — SaaS Landing Page
  Modern SaaS landing page for task and project management platforms built with Vite, Tailwind CSS, and JavaScript.
  
##  Live Demo
  https://task-flow-xi-eight.vercel.app/

##  Overview
  TaskFlow is a modern SaaS landing page designed to showcase a task and project management platform.
  The goal of this project was to create a clean, professional, and fully responsive user experience while following modern frontend development practices.
  The design focuses on performance, accessibility, usability, and conversion-oriented UI patterns commonly used in SaaS products.

---

##  Features

  - Modern SaaS landing page
  - Fully responsive design
  - Dark UI theme
  - Interactive navigation
  - Hero section
  - Features section
  - Pricing section
  - Testimonials section
  - FAQ section
  - Blog preview section
  - Call-to-action sections
  - Smooth scrolling experience
  - Mobile-friendly layout
